# AI Agent 개발을 위한 통합 Python 학습 계획

**총 연습 문제: 12개 핵심 문제**

## 1단계: Python 기초 강화 (2-3주)

### 핵심 개념
- **클래스와 객체지향 프로그래밍**
  - 클래스 정의, 인스턴스, 메서드
  - 상속, 다형성, 캡슐화
  - 매직 메서드 (`__init__`, `__str__`, `__call__`)
- **데코레이터**
  - 함수 데코레이터 이해와 구현
  - 클래스 데코레이터
  - property 데코레이터
  - functools.wraps 활용
